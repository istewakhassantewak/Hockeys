# 🏒 Hockeys - Landing Page

A modern and responsive hockey-themed landing page built using **HTML** and **Tailwind CSS**.  
Designed with clean UI, smooth layout, and mobile-first responsiveness.

🌐 **Live Demo:** https://hockeys-flax.vercel.app  

---

## 🚀 Features

- 🎯 Fully responsive design (mobile, tablet, desktop)
- 🧭 Clean navigation bar with responsive menu
- 🏑 Club / featured section layout
- 🎨 Modern UI using Tailwind CSS utility classes
- ⚡ Fast loading and optimized structure
- 📱 Mobile-first design approach

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS**
- **DaisyUI (if used)**
- **Vercel (Deployment)**
